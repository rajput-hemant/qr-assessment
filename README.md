## Getting Started

**Install dependencies:**

```bash
bun i || pnpm i || npm i
```

**Run the development server:**

```bash
bun dev || pnpm dev || npm run dev
```

**Run the Storybook:**

```bash
bun storybook || pnpm storybook || npm run storybook
```
